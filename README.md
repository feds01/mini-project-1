### Running the tests (without stacscheck)
- Ensure that ```lib/junit-platform-console-standalone.jar``` has executable permissions on linux or unix-like
  system by adding the 'execute' flag:  
  ```~/code/ $ chmod +x ./lib/junit-platform-console-standalone.jar```

- Compile the program (Linux):  
    ```~/code/ $ javac --cp "./lib/junit-platform-console-standalone.jar:./src/" src/test/Tests.java```
- Compile the program (Windows):  
    ```~/code/ $ javac --cp "./lib/junit-platform-console-standalone.jar;./src/" src/test/Tests.java```
- Run the compiled code:  
    ```~/code/ $ java -ea -jar ./lib/junit-platform-console-standalone.jar -cp src --scan-class-path```
